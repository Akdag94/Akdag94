## 🧑‍💻 About me

- 🏗️ I design, build and **operate** production web products end to end — from the first design frame to the nginx config serving it
- 🚀 Currently building **[Plankur](https://plankur.com)** — appointment scheduling SaaS for service businesses
- 🌍 Based in **Bursa, Türkiye**
- ⚙️ Everything I ship runs on infrastructure I manage myself — VPS, nginx, PM2, Cloudflare
